# Domain MOC

The ACP glossary — one page per business concept. Every name used in code or wiki
must resolve to a page here (Constitution Invariant 4).

- [[Worker]] — an actor that performs or supervises work.
- [[Workspace]] — the logical environment where work happens; the state-ownership unit.
- [[WorkUnit]] — a unit of work with a lifecycle state machine.
- [[Lease]] — a temporary, expiring claim over a resource.
- [[Artifact]] — a durable output of work.
- [[Checkpoint]] — a resumable summary of partial progress.
- [[Memory]] — durable workspace context for agent handoff and recall.
- [[Review]] — an explicit approval / rejection gate.
- [[Event]] — an immutable append-only record; the coordination primitive.
